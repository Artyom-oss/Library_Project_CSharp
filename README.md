LIBRARY CONSOLE APP (C#)

ОПИСАНИЕ ПРОЕКТА:
Это консольное приложение на языке C#.
Система библиотеки для работы с книгами и журналами.

ФУНКЦИОНАЛ:
- Добавление книги
- Добавление журнала
- Просмотр всех элементов библиотеки
- Удаление элемента по ID
- Просмотр статистики (книги, журналы, всего)

АРХИТЕКТУРА ПРОЕКТА:

MODELS:
- LibraryItem (базовый класс)
- Book (книга)
- Magazine (журнал)

REPOSITORY:
- Хранение данных в памяти (List)
- Add
- Remove
- GetAll

SERVICE:
- Бизнес-логика
- AddBook
- AddMagazine
- Remove
- GetAll
- GetStats

PROGRAM:
- Консольное меню
- Ввод данных пользователя
- Вызов сервисов

МЕНЮ ПРОГРАММЫ:
1. Добавить книгу
2. Добавить журнал
3. Посмотреть хранилище
4. Удалить
5. Статистика
0. Выход
